Exercise 5.

Use the exponential formulas to determine each of the following limits: \(\lim\limits_{t\to-\infty}\fe{\cosh}{t}\text{,}\) \(\lim\limits_{t\to\infty}\fe{\cosh}{t}\text{,}\) \(\lim\limits_{t\to-\infty}\fe{\sinh}{t}\text{,}\) and \(\lim\limits_{t\to\infty}\fe{\sinh}{t}\text{.}\)
Answer.
\(\lim\limits_{t\to-\infty}\fe{\cosh}{t}=\lim\limits_{t\to\infty}\fe{\cosh}{t}=\lim\limits_{t\to\infty}\fe{\sinh}{t}=\infty\text{;}\) \(\lim\limits_{t\to-\infty}\fe{\sinh}{t}=-\infty\)
Solution.
Each of the hyperbolic limits can be found quickly using these four basic limits: \(\lim\limits_{t\to-\infty}e^t=0\text{,}\) \(\lim\limits_{t\to\infty}e^t=\infty\text{,}\) \(\lim\limits_{t\to-\infty}e^{-t}=\infty\text{,}\) \(\lim\limits_{t\to-\infty}e^{-t}=0\text{.}\)
\begin{align*} \lim_{t\to-\infty}\fe{\cosh}{t}&=\lim_{t\to-\infty}\frac{e^{t}+e^{-t}}{2}&\lim_{t\to\infty}\fe{\cosh}{t}&\lim_{t\to\infty}\frac{e^{t}+e^{-t}}{2}\\ &=\infty&&=\infty\\ \lim_{t\to-\infty}\fe{\sinh}{t}&=\lim_{t\to-\infty}\frac{e^{t}-e^{-t}}{2}&\lim_{t\to\infty}\fe{\sinh}{t}&\lim_{t\to\infty}\frac{e^{t}-e^{-t}}{2}\\ &=-\infty&&=\infty \end{align*}
